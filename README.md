# Посмотри в окно

Проектная работа по курсу «Веб-разработчик» Яндекс Практикума.

## Описание проекта

Приложение позволяет искать видеоролики по городу и времени суток, просматривать найденные видео и переключаться между карточками.

## Технологии

* HTML
* CSS
* JavaScript

## Запуск проекта

Открой файл `index.html` в браузере или запусти проект через Live Server в VS Code.
